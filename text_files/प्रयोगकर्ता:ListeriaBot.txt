Experimental bot by
Magnus Manske
. It generates and updates lists on Wikipedia based on Wikidata.
Example
.
Use the local version of
Template:Wikidata list
and
Template:Wikidata list end
to define a list on a page.
Lists are based on SPARQL queries at
Wikidata Query Service
. For help about these, see
Wikidata Query Help
. You can request a query to be written for you at
Request a query
.
