Not done

Template documentation[view] [edit] [history] [purge]
लघु पथ:
Template:Notdone

This is an image-based template may be used on talk pages to show clearly that a section of discussion has not been resolved, so that each editor does not have to completely re-read the section.

Usage

[सम्पादन करी]

You may either use {{not done}} by itself for the default message or you may add a custom message as an optional parameter.

Template Result
{{not done}} Not done
{{not done|Custom message}} Custom message

Example

[सम्पादन करी]

आकृति:Pseudoheading I think this article needs more pictures. --User:John 00:00, 1 January 2000

I agree, especially the part about the estate grounds. --User:Jane 00:05, 1 January 2000
Can anyone find any, I've looked on Commons but there doesn't appear to be any. --User:George 00:23, 1 January 2000
Not done The only pictures I can find are copyrighted, can anyone else help? --User:John 00:30, 1 January 2000
Can anyone take some photos?. --User:Jane 01:00, 1 January 2000
 सम्पन्न I've uploaded one I took myself. --User:John 03:00, 1 January 2000
Thank you, that looks great. --User:George 12:00, 1 January 2000
Good bye!! --User:John 00:00, 2 January 2000

Multi-sign templates

[सम्पादन करी]

The following templates implement several icons:

  • {{AIV}} – used by Wikipedia:Administrator intervention against vandalism
  • {{EP}} – used for servicing requests in edit-protected areas
  • {{RFPP}} – used by Wikipedia:Requests for page protection
  • {{UAA}} – used by Wikipedia:Usernames for administrator attention
  • {{Smiley}} – used for informal communication purposes

Others

[सम्पादन करी]
  • Template:Resolved/See also, the smaller family of thread-level hatnote templates, similar to this but with a box around them. Any template above can be converted to one of those with {{Resbox}}
  • Template:Table cell templates/doc, the family of table-specific templates that work only in tables
  • Category:Image with comment templates
  • Category:Resolution templates
